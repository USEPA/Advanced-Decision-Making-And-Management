Before Start - Input and Output

ADAM is a tool that helps you build and solve supply chain management and design models. The participants of a supply chain include consumers, suppliers, technology providers, and transportation providers.

To construct a design model, one needs the following input data:

  • • Supplier Data: Costs and capacities for suppliers.
  • • Consumer Data: Costs and capacities for consumers.
  • • Tech Data: Costs and capacities for technology providers.
  • • Transport Data: Costs and capacities for transportation providers.

Given input data, a design model is solved to determine locations for new technologies and transactions of products that maximize the collective profit of all participants and that satisfy a given design budget.

A design model provides the following output data:

  • • Flow Data: Flows supply, consumed, and processed at different locations as well as transport between locations.
  • • Value Data: Value of products at different locations.
  • • Design Data: Optimal location for new technologies.

Once you have created a model, you can save it so that this can be duplicated and loaded to change features.
